What is Raspberry Pi?

Raspberry Pi is a mini server-PC that runs Raspberry Pi OS(linux that is customized) and meant for doing projects like:

  • Minecraft mods
  • Python Scripts
  • Creating a MP3 player
  • Hack radio waves

This PC runs ARM CPU that is 2.60 GHz speed. The main spefications about this PC are:

  • 8GB RAM
  • 2 USB 2.0 Ports
  • 2 USB 3.0 Ports
  • 2 micro HDMI Ports
  • 64-bit quad-core Cortex-A76 CPU
  • 1 Ethernet Port

What is recommended for your Raspberry Pi?

The main recommendations for your Raspberry PI are:

  • Cooler(sometimes you don't need to buy it)
  • Raspberry Pi Case
  • Raspberry Pi Power Supply(27W)
  • microSD Card
  • microHDMI -> HDMI Cable
  • Wired keyboard(or wireless)
  • Wired Mouse(or wireless)

How to setup Raspberry Pi?

Step 1

  1. Use a microSD card
  2. Put microSD Card at the port, that is in the back of your Raspberry Pi.
  3. Connect keyboard, mouse and microHDMI -> HDMI cable


Step 2

  1. Boot your PC
  2. Follow instructions on your screen
  3. Make a new user in setup


















Step 3

  1. Go to application menu
  2. Click on Preferences
  3. Install Visual Studio Code to begin coding


















Step 4

  1. Create a folder labled "Projects" in Documents
  2. Open Visual Studio Code and then open that folder
  3. Download Python in "Extentions" and create in Visual Studio Code a file labled "main.py"
